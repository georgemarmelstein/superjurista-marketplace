# SuperJurista Marketplace

Marketplace de plugins juridicos para Claude Code.

## Plugins disponiveis

| Plugin | Descricao | Versao |
|--------|-----------|--------|
| [superjurista-dev](https://github.com/georgemarmelstein/superjurista-dev) | Meta-ferramentas para criar sistemas agenticos judiciais | 1.0.0 |

## Como usar

### 1. Adicionar o marketplace

```
/plugin marketplace add georgemarmelstein/superjurista-marketplace
```

### 2. Instalar o plugin desejado

```
/plugin install superjurista-dev@superjurista-marketplace
```

### 3. Instalar o SuperJurista num projeto

```
/instalar-superjurista
```

## Sobre

Este marketplace reune plugins desenvolvidos para auxiliar magistrados e profissionais do direito no processamento de demandas judiciais usando Claude Code.
